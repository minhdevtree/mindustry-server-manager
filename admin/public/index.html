Server Console

Type a command above and press Enter. Common commands: status - server status help - list all commands players - online players config - show all config maps [type] - available maps (all/custom/default) save <slot> - save game to slot load <slot> - load save from slot saves - list all saves say <msg> - broadcast message kick <name> - kick player ban <type> <value> - ban player (name/id/ip) unban <ip/id> - unban player admin add/remove <name> - manage admins info <name/ip/uuid> - player info search <name> - search players fillitems [team] - fill core with items runwave - trigger next wave playerlimit [n/off] - set player limit rules - list game rules nextmap <name> - set next map shuffle [mode] - map shuffle mode
